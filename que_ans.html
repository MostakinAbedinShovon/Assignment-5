Question-1: What are the different ways to select an element in the DOM?

Elements in the DOM can be selected using methods like getElementById,getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll, or by traversing the DOM hierarchy.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML retrieves or sets the HTML markup inside an element, innerText returns or sets the visible text considering CSS styles, and textContent retrieves or sets all text within an element, ignoring formatting and hidden content.

Question-3: What is event delegation in the DOM?

Event delegation allows a parent element to handle events on its child elements by leveraging event propagation, specifically the bubbling phase, for efficient and dynamic event management.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a mechanism where an event triggered on a child element propagates upward through its ancestors, reaching the root of the document unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create elements using document.createElement, add them using methods like appendChild or insertBefore, and remove them using removeChild or the remove() method.
